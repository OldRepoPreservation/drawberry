DrawBerry Help

Contextual Toolbar

DrawBerry has a contextual toolbar over the canvas, let's explain the buttons.

Duplicate and delete buttons

Use these buttons to duplicate or delete the selected shapes.


Position buttons

Use these buttons to change the relative position of the selected shapes.


Transformation buttons

When selecting a rectangle or an oval click on these buttons to transform the shape into a polyline or a bezier curve.


Add and delete knob buttons

When editing a polyline or a bezier curve, use these buttons to add knobs between selected knobs or remove selected knobs.

Note :

You can also add a knob by alt-clicking near the line or the curve or by pressing the "a" key.


Replace buttons

When editing a polyline or a bezier curve, use these buttons to replace the part of the path between two selected knobs.


Boolean operations button

Use this button to perform boolean operations (union, difference, intersection and exclusive or) when selecting two shapes.
