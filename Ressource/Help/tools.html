DrawBerry Help

Tools

To draw with DrawBerry, you can use 6 different tools.

Choosing a tool

To choose a tool click on one of the button. By default, when you will have finish with this tool - by creating a shape, the tool selection will immediately jump to the selection tool. To "lock" on the selected tool, double-click on the button.
You can change this behavior in DrawBerry's preferences.


Selection tool

With the selection tool, you can select shapes and enter in the edit mode by double-clicking on a shape. To resize a selected shape, select a knob and drag it to the wished size
This is the default tool.

Modifier Keys

  • : Add/Remove shape of the current selection.
  • ^ : Rotation mode.
  • : Keep the ratio when resizing.


Rectangle tool


Use this tool to create rectangles and rounded rectangles (double-click to set the corner radius)

Modifier Keys

  • : Invert magnetic grid setting.


Oval tool


Use this tool to create ovals

Modifier Keys

  • : Invert magnetic grid setting.


Polyline tool


Use this tool to create polylines. Click to add a point, double-click and ^-click to add a last point or re-click on the first knob to close the line.

Modifier Keys

  • : Invert magnetic grid setting.


Bezier Curve tool


Use this tool to create polylines. Click to add a point, double-click and ^-click to add a last point or re-click on the first knob to close the line.

Modifier Keys

  • : Invert magnetic grid setting.
  • : Change control point angle.
  • ^ : Make control point symetric.
