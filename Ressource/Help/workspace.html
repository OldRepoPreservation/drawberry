DrawBerry Help

Working Space

DrawBerry's interface contains differents window. There all described here.

Document window

This is the main window. All the drawing will appear in this window.
Over the canvas, the contextual toolbar will provide you tools to arrange your drawing.


Tool panel

The upper part let you chose the used tool to draw. See the tool page for more informations.
The lower part gives a rapid access respectively to the document inspector, the shape inspector, the layer panel and the magnifying glass.


Layers panel

Use this window to create, delete and select layers. You can also lock and hide layers. Set the blending mode and the layer opacity.
Use the buttons to move the layers position.


Undo History

This window lists all the changes you have made to the document. Use the two left buttons to undo/redo changes and the right button to delete the history.


Inspectors


Use these pallettes to change the properties of the selected object or of the page.


Color Swatches

With this window, you can have a list of all the useful colors and then drop them on the shapes.


Color Swatches

This window provides you a library to manage shape collections of useful shapes
